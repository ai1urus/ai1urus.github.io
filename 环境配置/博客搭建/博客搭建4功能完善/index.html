添加评论系统

  1. 采用 valine 评论系统，具体步骤如下
  2. 注册 Leancloud 账号 官网地址
  3. 进入控制台，新建一个应用，名字任意
  4. 进入新创建的应用，点击 设置→应用 Key ，复制 App ID 和 App Key
  5. 点击 设置→安全中心 ，在Web安全域名一栏填入网站的域名
  6. 打开主题配置文件，定位到 valine ，将enable设为true，并将appid和appkey填入
  7. 部署完成后即可在文章下方看到评论区
