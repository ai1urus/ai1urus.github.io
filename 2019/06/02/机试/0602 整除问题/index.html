【牛客网】整除问题

https://www.nowcoder.com/questionTerminal/8e29045de1c84d349b43fdb123ab586a

问题描述

给定n，a求最大的k，使n！可以被a^k整除但不能被a^(k+1)整除。

输入

两个整数n(2<=n<=1000)，a(2<=a<=1000)

输出

一个整数.

样例输入

6 10

样例输出

1

解题思路

n！与 a^k 可能非常大，不可以直接求余。所以要从整除的特征入手，如果一个整数a可以整除整数b，分别对a和b分解素因数可得：
$$
a=p_1^{e_1}p_2^{e_2}…p_n^{e_n}
$$

$$
b=p_1’^{e_1’}p_2’^{e_2’}…p_n’^{e_n’}
$$

由b除以a可得：
$$
\frac{b}{a}=\frac{p_1’^{e_1’}p_2’^{e_2’}…p_n’^{e_n’}}{p_1’^{e_1’}p_2’^{e_2’}…p_n’^{e_n’}}
$$
如果a能整除b，则b/a为一个整数，如果a存在一个素因数px，则b也一定有该素因数，且b中该素因数的幂指数必不小于a中对应的幂指数。

现在我们求的是n! 与 a^k 整除的结果，a^k可以写成：
$$
a^k=(p_1^{e_1}p_2^{e_2}…p_n^{e_n})^k
$$

$$
a^k=p_1^{e_1k}p_2^{e_2k}…p_n^{e_nk}
$$

所以我们最终就是要求出最大的非负整数k，使a中任一素因数的幂指数的k倍仍小于或等于该素因数在n！中对应的幂指数。

对a分解质因数可以使用常规的方法，但是n！比较大，不可以直接分解，王道宝典上给出了一种比较巧妙的方法，步骤如下：

  1. 对于每一个素因数p，设置一个计数器，该计数器记录n！中p的幂次
  2. 计算n/p，将其累加到技术器中。因为n！中每有一个p的倍数，p的幂次就会加1。
  3. 计算n/(p*p)，将其累加到计数器中。n！中每有一个p * p的倍数，p的幂次就会加2，但由于之前已经加过一次，所以这里加1。
  4. 依次计算n/(p * p * p)以及p的更高次幂，直至n/(p^k)等于0，表示没有更多的数可以提供p银子。
  5. 依次遍历所有可能成为n！的素因数的素数（即小于等于n的所有素数），并计算其幂指数，即可完成对n！的素因数分解。

代码

#include<iostream>
#include<math.h>
#include<memory.h>
using namespace std;

unsigned int counta[1001];
unsigned int countn[1001];
// 判断是否为质数
bool isZhi(unsigned int num) {
	if (num == 2) return true;
	if (num == 1 || num == 0 || num < 0) return false;
	unsigned int bound = sqrt(num) + 1;
	for (unsigned int i = 2; i <= bound; i++) {
		if (num % i == 0) return false;
	}
	return true;
}
// 采用常规方法对A进行质因数分解
unsigned int FenA(unsigned int a, unsigned int* count) {
	for (unsigned int i = 2; i <= a ; i++) {
		while (a % i == 0) {
			a /= i;
			count[i]++;
		}
	}
	return 0;
}
// 特殊方法对n！进行素因数分解
unsigned int FenN(unsigned int n, unsigned int* count) {
	for (unsigned int i = 2; i <= n; i++) {
		if (isZhi(i)) {
			unsigned int tmpi = i;
			while (n / tmpi > 0) {
				count[i] += (n / tmpi);
				tmpi *= i;
			}
		}
	}
	return 0;
}
// 找到最大的k
unsigned int FindK(unsigned int* counta, unsigned int* countn) {
	unsigned int ans = 1000;
	for (unsigned int i = 0; i < 1001; i++) {
		if (isZhi(i) && counta[i]>0 && countn[i]>0) {
			unsigned int tmp = countn[i] / counta[i];
			if (tmp < ans) {
				ans = tmp;
			}
		}
	}
	return ans;
}

unsigned int main() {
	unsigned int n, a;
	while (cin >> n >> a) {
		memset(counta, 0, sizeof(counta));
		memset(countn, 0, sizeof(countn));
		FenA(a, counta);
		FenN(n, countn);
		cout << FindK(counta, countn) << endl;
	}
}
