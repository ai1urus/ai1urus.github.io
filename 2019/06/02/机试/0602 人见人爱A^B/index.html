【HDU2035】人见人爱A^B

http://acm.hdu.edu.cn/showproblem.php?pid=2035

问题描述

求A^B的最后三位数表示的整数。
说明：A^B的含义是“A的B次方”

输入

输入数据包含多个测试实例，每个实例占一行，由两个正整数A和B组成（1<=A,B<=10000），如果A=0, B=0，则表示输入数据的结束，不做处理。

输出

对于每个测试实例，请输出A^B的最后三位表示的整数，每个输出占一行。

样例输入

2 3
12 6
6789 10000
0 0

样例输出

8
984
1

思路

水题，主要锻炼一下二分求幂思想的应用。

二分求幂即将幂次转化成二进制，每次循环将底数自乘，如果幂次对应的二进制位数为1，则在结果中加入当前的底数。

以2^15为例：

底数 幂次%2 ans操作 ans结果 幂次操作 幂次结果
2 15%2==1 1*=2 2 15/=2 7
4 7%2==1 2*=4 8 7/=2 3
16 3%2==1 8*=16 128 3/=1 1
256 1%2==1 128*=256 32768 1/=2 0

代码

#include<iostream>
#include<math.h>
using namespace std;

unsigned int main() {
	unsigned int A, B;
	while (cin >> A >> B) {
		if (A == B && A == 0) break;
		unsigned int ans = 1;
		while (B > 0) {
			if (B % 2 == 1) {
				ans *= A;
				ans %= 1000;
			}
			B /= 2;
			A *= A;
			A %= 1000;
		}
		cout << ans << endl;
	}
}
