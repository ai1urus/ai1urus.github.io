【牛客网】最大公约数

问题描述

输入两个正整数，求其最大公约数。

输入描述

测试数据有多组，每组输入两个正整数。

输出描述

对于每组输入,请输出其最大公约数。

示例输入

49 14

示例输出

7

问题思路

设两个数为A、B，其最大公约数为g，则：（k与l均为整数）
$$
A = g * k
$$

$$
B = g * l
$$

假设A >= B，则：（m与r均为整数）
$$
A = B*m + r
$$
即
$$
r = g * ( k - l * m )
$$
因为k-l*m为整数，所以r取余g也为0，问题就转变为求 r和B 的最大公约数，如此迭代下去。

当最终r = 0时即求得最大公约数

代码

#include<iostream>
using namespace std;

int main(){
    int A,B;
    while(cin>>A>>B){
        if(A==0 && B!=0) cout<<B<<endl;
        else if(A!=0 && B==0) cout<<A<<endl;
        else{
            int tmp;
            do{
                int tmp = A;
                A = B;
                B = tmp % B;                
            }while(B!=0);
            cout<<A<<endl;
        }
    }
}
